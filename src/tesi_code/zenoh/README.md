```install_zenoh.sh```

install also the zenoh bridge using script
```dowload_zenoh_bridge.sh``` in the folder do you want to have zenoh

### on client

install also the zenoh bridge using script
```dowload_zenoh_bridge.sh``` in the folder do you want to have zenoh
